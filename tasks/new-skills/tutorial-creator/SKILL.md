# Tutorial Creator

## When to use
Create a structured, beginner-friendly tutorial on any technical topic.

## Workflow
1. **Define audience** — skill level, prerequisites, goal.
2. **Outline** — numbered steps with expected outcome per step.
3. **Write** — each step includes:
   - What you'll do
   - Code or commands (with syntax highlighting)
   - Expected output
   - Common pitfalls
4. **Add visuals** — screenshots or diagrams (described in alt text).
5. **Test** — execute all commands to verify they work.
6. **Output** — markdown file with complete tutorial.

## Output
A tested, step-by-step tutorial in markdown.
