# Deadline Tracker

## When to use
Scan project files for deadlines and generate a timeline or reminder list.

## Workflow
1. **Scan** — search files (markdown, yaml, json, text) for date patterns.
2. **Parse** — extract dates with context (what the deadline is for).
3. **Sort** — chronological order, flag items due within 7 days.
4. **Output** — timeline table or calendar view.
5. **Reminders** — optionally create a reminders file for import.

## Output
Deadline report sorted by urgency with context for each date.
