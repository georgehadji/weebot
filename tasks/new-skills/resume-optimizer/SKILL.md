# Resume Optimizer

## When to use
Tailor a resume or CV to match a specific job description.

## Workflow
1. **Read resume** — parse the current resume (PDF, docx, or markdown).
2. **Read job description** — extract required skills, keywords, and qualifications.
3. **Gap analysis** — identify which required skills are missing or underemphasized.
4. **Rewrite** — optimize sections:
   - Summary to match role
   - Skills section to include missing keywords
   - Experience bullets to quantify achievements
   - Remove irrelevant content
5. **ATS check** — verify keyword match rate and format compatibility.
6. **Output** — optimized resume with a summary of changes.

## Output
An ATS-optimized resume tailored to the target job.
