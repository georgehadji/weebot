# Proposal Writer

## When to use
The user needs a structured proposal document — business proposal, project plan, or grant application.

## Workflow
1. **Gather requirements** — ask about: problem statement, audience, budget, timeline, format.
2. **Structure the proposal:**
   - Executive summary
   - Problem statement
   - Proposed solution
   - Timeline and milestones
   - Budget estimate
   - Team/qualifications
   - Risks and mitigations
3. **Draft each section** with clear, professional language.
4. **Review** — check for consistency, completeness, and persuasive framing.
5. **Output** — write as markdown or docx.

## Output
A complete proposal document.
