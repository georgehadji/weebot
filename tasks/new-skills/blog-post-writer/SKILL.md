# Blog Post Writer

## When to use
Write a complete, well-researched blog post on any topic.

## Workflow
1. **Research** — search the web for current information on the topic.
2. **Outline** — structure with hook, intro, body sections, conclusion, CTA.
3. **Write** — generate the post in markdown with:
   - SEO-optimized title and meta description
   - Engaging introduction
   - 3-5 body sections with subheadings
   - Credible references and links
   - Clear conclusion with call to action
4. **Optimize** — check readability, keyword density, and suggested tags.
5. **Output** — save as markdown file ready for publishing.

## Output
A complete blog post in markdown with SEO metadata.
