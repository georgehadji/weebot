# Newsletter Composer

## When to use
Curate and compose an email newsletter from multiple sources.

## Workflow
1. **Gather content** — pull from provided links, web research, or a content feed.
2. **Curate** — select 3-5 most relevant items per section.
3. **Write** — for each item: compelling headline, 1-2 sentence summary, link.
4. **Structure** — header, featured story, regular items, sponsor (if any), footer.
5. **Subject line** — 3 options with A/B testing rationale.
6. **Output** — markdown or HTML ready for email sending.

## Output
A complete newsletter with curated items and subject line options.
