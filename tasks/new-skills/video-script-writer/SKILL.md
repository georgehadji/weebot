# Video Script Writer

## When to use
Write a timed script for educational videos, demos, or presentations.

## Workflow
1. **Define format** — tutorial, product demo, explainer, or presentation.
2. **Structure** — intro (hook + agenda), body (3-5 segments), outro (summary + CTA).
3. **Write script** with:
   - Timestamp per section
   - Narration text
   - Visual cues (what to show on screen)
   - Transition notes
4. **Estimate duration** — based on word count (~150 words/minute).
5. **Output** — script in markdown with timestamp column.

## Output
A timed video script with narration and visual cues.
