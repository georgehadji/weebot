PowerShell & WSL2 command execution

Execute native PowerShell 5.1 and WSL2 bash commands directly on Windows 11 with full compatibility, real-time output streaming, and security sandboxing.

Browser automation with Playwright

Automate web interactions using Playwright for scraping, testing, and agentic browsing with headless or headed modes and screenshot capabilities.

File editing with safety guards

Safely edit, create, and manage files using guarded operations with preview, diff, and rollback support to prevent accidental data loss.

Python code execution in sandboxed subprocesses

Execute Python scripts in isolated subprocesses with support for pandas, numpy, matplotlib, and controlled environment variables.

Real-time event streaming via WebSocket

Stream live logs, tool outputs, and agent events over WebSocket for responsive UIs and real-time monitoring of all agent activity.

MCP server for IDE integration

Expose Model Context Protocol server for seamless integration with IDEs like VS Code, Cursor, and Windsurf for agentic coding workflows.

Multi-model AI support (OpenRouter, Anthropic, DeepSeek, OpenAI)

Route requests across multiple LLM providers including OpenRouter, Anthropic, DeepSeek, and OpenAI with intelligent fallback, cost optimization, and per-model tool routing.
