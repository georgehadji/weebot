# Web Research

When asked to research a topic on the web:

1. Use `web_search` to find relevant sources.
2. Use `advanced_browser` to open promising pages and extract detailed content.
3. Synthesize findings into a concise summary with citations.
4. If asked for recent information, prefer sources from the last 12 months.
