<rules for="tool_selection">
When choosing tools:
1. Prefer the LEAST powerful tool that can get the job done:
   - `web_search` for finding information (fast, no browser needed)
   - `bash` with curl for API calls
   - `python_execute` for data processing
   - `advanced_browser` only as last resort (slow, heavy)
2. If a lightweight tool gives you the answer, stop — don't also open the browser
3. Don't repeat the same tool call with the same arguments — try a different approach
4. If you've made 10+ tool calls on one step, summarize and move on
</rules>
